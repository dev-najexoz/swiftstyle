Why Choose SwiftStyle?

SwiftStyle is a lightweight, responsive, and powerful CSS utility library. It offers a simple way to create well-aligned and modern web designs without much hassle.

Key Features

  • Responsive containers for all screen sizes
  • Predefined box styling with borders, padding, and shadows
  • Easy-to-use flexbox and grid layout utilities
  • Quick alignment and spacing utilities
  • Simple text styling options

How to Use SwiftStyle

<link rel="stylesheet"https://dev-najexoz.github.io/swiftstyle/style.css">

This single line of code will link SwiftStyle to your HTML file. Then, you can start using the utility classes directly in your HTML elements.
